API Документация

Эндпоинты

  • POST /create-user - создание пользователя. Требуется JSON тело с полями 'username' и 'password'.
  • POST /authenticate - аутентификация пользователя. Требуется JSON тело с полями 'username' и 'password'.
  • POST /add-product - добавление товара. Требуется JSON тело с информацией о товаре.
